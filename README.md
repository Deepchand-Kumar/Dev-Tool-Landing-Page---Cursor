# Dev-Tool-Landing-Page---Cursor
This project recreates a desktop-first developer tool landing page inspired by the Cursor website.  


Title : Cursor-Inspired Developer Tool Landing Page Clone (Assignment)


This project recreates a desktop-first developer tool landing page inspired by the Cursor website.  

The goal of this assignment was to closely match the original design in terms of:

- Page structure
- Layout and spacing
- Typography
- Color palette
- Visual hierarchy

No animations, no responsiveness, and no JavaScript were used — only HTML and CSS.


Tech Stack

- HTML5
- CSS3
- No JavaScript
- No frameworks
- Desktop-only layout

The following sections were recreated based on the original Cursor website layout:

1. Top Navigation Bar  
   - Logo  
   - Navigation links  
   - Primary CTA button  
   - Dark background  

2. Hero Section  
   - Large headline  
   - Supporting description  
   - Primary CTA  
   - Product screenshot  

3. Trusted By Section  
   - Horizontal row of company logos  

4. Feature Sections (3 Blocks)  
   - Two-column layout  
   - Alternating image and text alignment  

5. Feature Cards Section  
   - Section heading  
   - Grid layout (3–4 feature cards)  

6. Testimonials  
   - Quote cards  
   - Name and role of users  

7. Use Cases / Stories  
   - Image + short text cards  

8. Changelog / Updates  
   - List-style update layout  
   - Dates included  

9. Team / About Section  
   - Large image  
   - Short description  
   - CTA button  

10. Final CTA Section  
    - Large heading  
    - Single prominent button  

11. Footer  
    - Multi-column layout  
    - Company info  
    - Navigation links  

    Color Palette

Primary Background: #0D0D0D  
Secondary Background: #111111  
Accent Color: #5B5BFF  
Text Primary: #FFFFFF  
Text Secondary: #B3B3B3  
Card Background: #1A1A1A  
Border Color: #2A2A2A
